# 新增功能简介

- 支持顶部、左侧和右侧屏幕边缘吸附及自动隐藏。
- 顶部折叠时显示横向额度摘要；左右折叠时显示竖向 5 小时剩余进度。
- 侧边进度从底部向顶部填充，百分比数字随填充端点移动。
- 隐藏和弹出使用统一、可中断、可反向的平滑动画。
- 支持记忆普通窗口位置，以及顶部、左侧和右侧的展开或折叠状态。
- 支持多显示器、负坐标显示器与 Windows DPI 缩放。
- 支持立即刷新、自动刷新及失败后保留上一次有效数据。
- 窗口置顶、可拖动、圆角显示，并且不占用任务栏位置。
- 根据实时天气的雨、雾、雪或雷暴状态显示对应环境效果。
- 默认提供明确标注 DEMO 的安全演示数据与界面。
- 不读取 auth.json，不保存 Token，不包含遥测或分析服务。
