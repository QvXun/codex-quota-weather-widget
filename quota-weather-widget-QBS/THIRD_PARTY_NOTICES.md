# 第三方与平台声明

## Windows 与 .NET 平台组件

本项目使用 Windows 自带的 PowerShell、WPF（PresentationFramework / PresentationCore / WindowsBase）及 Windows Forms API 创建悬浮窗和获取显示器工作区。它们随 Windows/.NET 平台提供，不随本仓库重新分发。

## Open-Meteo

启用天气且完成用户本地位置配置后，程序会向 api.open-meteo.com 请求当前天气。该服务条目仅说明运行时用途；公开发布前请自行核对其当前服务条款、使用限制与署名要求。

## 上游项目

当前可审查副本没有附带可验证的上游许可证或完整署名材料。公开上传前必须确认上游代码、图标、字体及其他资源的来源和许可，并在本文件中补充原始项目链接、版权声明与适用许可证。
